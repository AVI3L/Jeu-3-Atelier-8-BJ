Regles des jeux

Cette page explique simplement le fonctionnement des trois projets disponibles dans le menu principal.

Roulette / Machine a sous

Le joueur lance une machine a sous avec trois rouleaux. Le but est d'obtenir les meilleurs symboles possibles pour gagner des pieces.

  • Le solde de depart peut etre regle avant de jouer.
  • Chaque lancer coute le montant indique dans la mise.
  • Le bouton de lancement ou le levier lance les trois rouleaux.
  • Le bouton de mise ajoute 10 pieces a la mise.
  • Le bouton de mise maximum place tout le solde disponible.
  • Trois symboles identiques donnent un gain selon le multiplicateur affiche dans le tableau des gains.
  • Deux symboles identiques remboursent la mise.
  • Une cloche presente sur un rouleau donne un petit bonus.
  • Si le solde devient insuffisant, une zone de recharge permet de remettre des pieces.

Memory Cards

Le joueur retourne des cartes pour retrouver toutes les paires. Le but est de terminer la grille avec le moins de coups possible.

  • Trois difficultes sont disponibles : facile avec 8 cartes, moyen avec 16 cartes et difficile avec 24 cartes.
  • Le joueur retourne deux cartes a chaque tour.
  • Si les deux cartes ont le meme symbole, la paire reste visible.
  • Si les cartes sont differentes, elles se retournent apres un court delai.
  • Le compteur de coups augmente a chaque carte retournee.
  • Le chronometre demarre avec la partie et s'arrete quand toutes les paires sont trouvees.
  • Le combo augmente quand plusieurs paires sont trouvees a la suite.
  • Le bouton Nouvelle partie relance le jeu avec la difficulte choisie.

Blackjack

Le joueur affronte la banque. Le but est d'avoir un score plus proche de 21 que la banque, sans depasser 21.

  • Le joueur et la banque recoivent chacun deux cartes au debut.
  • Les cartes numeriques valent leur valeur, les figures valent 10 et l'as vaut 11 ou 1 selon le score.
  • Le bouton Tirer ajoute une carte a la main du joueur.
  • Le bouton Rester termine le tour du joueur et laisse jouer la banque.
  • La banque tire automatiquement tant que son score est inferieur a 17.
  • Si le joueur depasse 21, il perd immediatement.
  • Si la banque depasse 21, le joueur gagne.
  • Si personne ne depasse, le score le plus eleve gagne. En cas d'egalite, la partie est nulle.
  • Un blackjack est obtenu avec un as et une carte valant 10 des le debut.
